is-object-empty
---------------

Check to see if an object is empty (has no keys)

install
-------

```bash
npm install is-object-empty
```

usage
-----

```js
var isObjectEmpty = require('is-object-empty');

var obj = {};

if (isObjectEmpty(obj)) {
    console.log('yup, it is empty');
}
else {
    console.log('nope, it is not empty');
}
```
